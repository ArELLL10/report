HAMILTONIAN GRAPH

• A HAMILTONIAN PATH IS A PATH IN A GRAPH THAT VISITS EACH VERTEX EXACTLY ONCE.
• THE PATH DOES NOT NEED TO BE A CYCLE, MEANING IT DOES NOT NECESSARILY RETURN TO THE STARTING VERTEX.
• A GRAPH CONTAINS A HAMILTONIAN PATH IF THERE EXISTS A SEQUENCE OF EDGES WHERE ALL VERTICES ARE VISITED EXACTLY ONCE.
