HAMILTONIAN CIRCUIT (CYCLE)

• A HAMILTONIAN CIRCUIT (OR HAMILTONIAN CYCLE) IS A SPECIAL TYPE OF HAMILTONIAN PATH WHERE, IN ADDITION TO VISITING EVERY VERTEX EXACTLY ONCE, THE PATH RETURNS TO THE STARTING VERTEX, FORMING A CLOSED LOOP.
• A GRAPH CONTAINS A HAMILTONIAN CIRCUIT IF THERE EXISTS A CYCLE THAT INCLUDES ALL THE VERTICES OF THE GRAPH EXACTLY ONCE, STARTING AND ENDING AT THE SAME VERTEX.
